
# LoRA Fine-Tuning Project

## Objective
Fine-tune a language model to convert informal text into formal English.

## Tools Used
- Python
- Hugging Face
- Unsloth
- PyTorch

## Steps
1. Created dataset (100+ examples)
2. Fine-tuned model using LoRA
3. Generated loss curve
4. Compared outputs

## Results
The fine-tuned model produces more formal responses compared to base model.

## Conclusion
LoRA helps efficiently fine-tune large models with minimal resources.
